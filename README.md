# StudentLogger.com - Domain Sales Page

A responsive single-page application built with vanilla HTML, CSS, and JavaScript to showcase and sell the premium domain **studentlogger.com**.

## Features

- **Single Page Application** - No build process needed, works out of the box
- **Responsive Design** - Mobile, tablet, and desktop optimized
- **Smooth Navigation** - Seamless scrolling between sections
- **Contact Form** - Allow interested buyers to submit inquiries
- **Value Proposition** - Clear messaging about why this domain is valuable
- **Modern UI** - Clean, professional design with gradient backgrounds
- **Zero Dependencies** - Pure HTML, CSS, and JavaScript

## Project Structure

```
studentlogger-domain/
├── index.html          # Main HTML file
├── css/
│   └── style.css      # All styling
├── js/
│   └── script.js      # Interactivity and form handling
└── README.md          # This file
```

## Getting Started

### View Locally

Simply open `index.html` in your web browser to view the site locally.

## Sections

1. **Navigation Bar** - Quick links to all sections
2. **Hero Section** - Eye-catching domain display
3. **About** - Description of the domain
4. **Value Proposition** - Key reasons why the domain is valuable
5. **Use Cases** - Different applications for the domain
6. **Contact Form** - Buyer inquiries
7. **Footer**

## Customization

Edit `index.html` for content and `css/style.css` for styling.
